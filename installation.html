Dashboards mit Shiny

The R Bootcamp @ CSS
März 2023




1 - Installiere Base-R und RStudio auf deinem Laptop.

Installiere mit Hilfe der IT die neusten verfügbaren Versionen von R und RStudio.

2 - Installiere zusätzliche Pakete

Wir werden verschieden R Pakete verwenden, die vorher installiert werden müssen. Bitte führe den folgenden Code in der R Console in RStudio aus:

install.packages(c( "shiny", "shinyFeedback", "waiter", "reactlog", "shinyloadtest", "thematic", "bslib", "shinythemes", "shinycssloaders", "shiny.collections", "viridis", "ggthemes", "ggrepel", "packcircles", "treemap", "ggstance", "tidyverse", "glue", "lubridate", "tools", "devtools", "usethis", "scales" ))

Wenn Du gefragt wirst, ob du Pakete from source installieren möchtest, gebe n ein und drücke Enter. Dies sollte es ermöglichen die Pakete ohne Admin-Rechte zu installieren.

3 - Optional: Zusätzliche Software installieren

Falls das Installationsskrip aus 2) Fehlermeldungen anzeigt (oder du Pakete from source kompilieren möchtest), installiere bitte folgende Software und lasse das Skript nochmal laufen. Installation dieser Software benötigt Admin-Rechte.

Windows: RTools
Mac: Command line tools (kein direkter Link verfügbar)

4 - Kursseite & Materialien

Unter diesem Link https://therbootcamp.github.io/Shiny_2023CSS findest Du die Kursseite mit allen Materialien.

Bitte lade vor Kursbeginn das Projekt herunter, entpacke es (Doppelklick), und bewegt den darin enthaltenen Ordner “TheRBootcamp” auf den Desktop oder in einen anderen Ordner Deiner Wahl.

5 - Eingangsfragebogen

Bitte nehmt Euch ein paar Minuten und füllt diesen Fragebogen aus. Er hilft uns den Kurs auf Euch zuzuschneiden.
